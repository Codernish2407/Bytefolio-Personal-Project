# **App Name**: ByteFolio

## Core Features:

- Dynamic Hero Section: Animated introduction with name, role, tagline, and CTA buttons for portfolio and resume.
- Database-Driven About Me: Displays education, interests, and focus areas fetched from MongoDB, editable via admin panel.
- Categorized Skills Showcase: Organizes skills into languages, frameworks, tools, and CS fundamentals with links to relevant projects.
- Comprehensive Project Portfolio: Details projects with title, description, tech stack, GitHub link, live demo, and category. Designed for case-study expansion.
- Experience and Internships: Presents roles, durations, impacts, and tools used in past experiences.
- Achievements & Hackathons: Displays hackathons, certifications, and competitive programming achievements.
- Backend-Powered Contact Form: Stores contact form submissions in MongoDB with optional email notifications. Includes logic that is powered by a tool that reasons about personally identifiable information.

## Style Guidelines:

- Color Palette: Dark background (#121212) for a modern feel.
- Primary color: Vibrant blue (#29ABE2) for a futuristic look.
- Accent color: Electric purple (#9400D3) to highlight key elements.
- Body and headline font: 'Inter' (sans-serif) for clean, readable text; particularly well suited for body text.
- Code font: 'Source Code Pro' (monospace) for code snippets.
- Glassmorphism cards for a sleek, modern interface.
- Subtle micro-animations for engaging interactivity.
